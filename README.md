# claude-plugins

A collection of [Claude Code](https://claude.ai/code) plugins.

## Getting started

```sh
curl -fsSL https://raw.githubusercontent.com/cgraf78/claude-plugins/main/install.sh | bash
```

## Plugins

- [status-line](./plugins/status-line/) — Rich status line showing host, directory, git, model, context, cost, and turns
