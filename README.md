# claude-plugins

A collection of [Claude Code](https://claude.ai/code) plugins.

## Getting started

```sh
curl -fsSL https://raw.githubusercontent.com/cgraf78/claude-plugins/main/install.sh | bash
```

## Plugins

- [status-line](./status-line/) — Rich status line showing host, directory, git, model, context, cost, and turns
- [memory-sync](./memory-sync/) — Sync project memory to a cloud drive for sharing across machines
